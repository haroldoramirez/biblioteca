1 Introdução

1.1 Objetivo

Este conjunto de páginas tem por objetivo apresentar as características e funcionalidades do Sistema Biblioteca Digital, com o intuito de possibilitar ao usuário um completo conhecimento do sistema, bem como para sanar dúvidas e dificuldades que venham a sugir durante a utilização do mesmo. Diante disso, serão apresentadas as telas do sistema, juntamente com seus respectivos aspectos e responsabilidades.

2 Biblioteca do Biogás

O sistema consiste em realizar o armazenamento de conteúdos sobre biogás, manter esses cadastros sempre atualizados e o sistema permite o cadastro de:

  • Cadastro de Trabalhos Acadêmicos
  • Cadastro de Artigos Científicos
  • Cadastro de Publicações
  • Cadastro de Eventos
  • Cadastro de Cursos
  • Cadastro de Vídeos
  • Cadastro de Notícias
  • Cadastro de Marcos Regulatórios
  • Cadastro de Sites
  • Cadastro de Albuns de Fotos
  • Cadastro de Usuários
